# Dasboard for E Commerce Application for windows

A windows based dasboard to manage the mobile application 

### Note: This dashboard and application was replaced by an improved version

## Features

- View active users
- Add new Products by categories
- View and manage current orders by Users

## Tools used

- Flutter
- Dart
- Firebase (Firestore)
